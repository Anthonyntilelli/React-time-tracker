# React-time-tracker

 React/Redux project for Flatiron.
 Project implements a subset of employee time tracking system.

## Environment
  - RAILS_JWT_ALG - Algorithm used for JWT, it defaults to HS256.
  - RAILS_JWT_SECRET_KEY - Secret key for JWT Tokens.
